## Постановка задачи
Разработать лексический анализатор (сканер), позволяющий выделить в тексте лексемы.
Встроить сканер в интерфейс текстового редактора.

____
## Персональный вариант задания
Номер темы в курсовой работе - 41:
Объявление и инициализация строковой константы на языке Pascal.  
Пример верной строки:
```Pascal
Const Stroka: string = 'Привет';
```
