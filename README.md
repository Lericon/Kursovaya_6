## Область ввода/редактирования текста
Данная область предназначена для ввода, редактирования и форматирования текста в программе.

В области ввода/редактирования текста можно выполнять следующие действия:

* __Ввод текста:__ Набирайте текст, который будет отображаться в этом поле.
* __Редактирование текста:__ Изменяйте уже введённый текст, используя стандартные операции
* __Выделение текста:__ Используйте мышь или клавишу Shift вместе с клавишами стрелок для выделения части текста.
* __Применение команд:__ Для работы с текстом можно использовать команды "Отменить", "Повторить", "Копировать", "Вырезать" и другие через контекстное меню или горячие клавиши.

Область ввода/редактирования текста является основной зоной для работы с документом, и все изменения происходят именно здесь. Чтобы сохранить изменения, используйте команду "Сохранить".

Примечание: Вы также можете использовать клавиши быстрого доступа (горячие клавиши) для ускорения работы с текстом.

## Область отображения результатов
Данная область предназначена для отображения результатов работы языкового процессора, таких как ошибки и предупреждения, найденные в тексте, введённом в область редактирования.

В этой области отображаются следующие данные:
* __Ошибки и предупреждения:__ Языковой процессор анализирует введённый текст и выделяет синтаксические ошибки.

__Особенности:__
* Вы этой области __невозможно вводить текст__. Она только отображает результаты работы программы.
* Текст в области изменяется только после нажатия кнопки "Пуск", которая запускает анализ текста и выводит соответствующие сообщения о найденных ошибках или проблемах.

Чтобы обновить результаты, необходимо снова запустить анализ с помощью кнопки __"Пуск"__.

## Панель инструментов
Панель инструментов позволяет использовать основные действия программы, с помощью кнопок, расположенных на ней.
Каждая кнопка соответствует определённой операции:
* __Создание файла:__ Создаёт новый пустой файл для редактирования.
* __Открыть файл:__ Открывает существующий файл для редактирования.
* __Сохранить:__ Сохраняет изменения в текущем файле.
* __Отменить:__ Отменяет последнее действие с текстом.
* __Повторить:__ Повторяет последнее действие с текстом.
* __Копировать:__ Копирует выделенный текст в буфер обмена.
* __Вырезать:__ Вырезает выделенный текст и помещает его в буфер обмена.
* __Вставить:__ Вставляет текст из буфера обмена в текущий документ.
* __Пуск:__ Вставляет текст из буфера обмена в текущий документ.
* __Справка:__ Открывает справочную информацию по использованию программы.
* __О программе:__ Открывает окно с информацией о программе и разработчике.

## Файл

### Создание файла
Данная функция позволяет создать новый пустой текстовый файл в программе.

При нажатии кнопки __"Создание файла"__ в меню программы создается новый документ с названием _"Новый документ"_. Однако файл не сохраняется автоматически, и для его сохранения необходимо выбрать соответствующую команду в меню.

После создания файла пользователь может ввести в него текст и при необходимости сохранить его вручную.

### Открыть
Данная функция позволяет открыть существующий текстовый файл для редактирования.

При нажатии кнопки __"Открыть"__ в меню программы появится диалоговое окно выбора файла. Пользователь может выбрать нужный файл, после чего его содержимое отобразится в окне программы.

После открытия файла его можно редактировать и при необходимости сохранить изменения.

### Сохранить
Данная функция позволяет сохранить изменения в текущем файле.

При нажатии кнопки __"Сохранить"__ в меню программы выполняется одно из следующих действий:
* Если файл уже существует, его содержимое будет сохранено без дополнительных запросов.
* Если файл не был сохранен ранее, появится диалоговое окно, где пользователь может выбрать место и указать имя для сохранения.

После сохранения пользователь может продолжить редактирование файла.

### Сохранить как
Данная функция позволяет сохранить текущий файл с новым именем или в другом месте.

При нажатии кнопки __"Сохранить как"__ в меню программы появится диалоговое окно, где пользователь может выбрать папку для сохранения и задать новое имя файла.

После сохранения программа продолжит работу с новым файлом, а исходный файл останется без изменений.

### Закрыть
Данная функция позволяет закрыть текущий файл.

При нажатии кнопки __"Закрыть файл"__ в меню программы выполняется одно из следующих действий:
* Если файл был сохранен, он просто закрывается.
* Если в файле есть несохраненные изменения, появится окно с вопросом __"Сохранить файл?"__.
	Пользователь может выбрать:
	* __Да__ - сохранить изменения перед закрытием.
	* __Нет__ - закрыть файл без сохранения.

### Выход
Данная функция позволяет закрыть приложение.

При нажатии кнопки __"Выход"__ программа выполнит следующие действия:
* Если все файлы сохранены, приложение закроется без дополнительных запросов.
* Если есть несохраненные файлы, для каждого из них появится окно с вопросом __"Сохранить файл?"__.
	Пользователь может выбрать:
	* __Да__ - сохранить изменения перед закрытием.
	* __Нет__ - закрыть файл без сохранения.

После того, как все несохраненные файлы будут обработаны, приложение будет закрыто.

## Правка

### Отменить
Данная функция позволяет отменить последнее выполненное действие с текстом.

При нажатии кнопки __"Отменить"__ программа возвращает текст к состоянию до последнего изменения. Это может быть удаление, вставка, редактирование текста и другие действия.

Функция может использоваться несколько раз подряд для поэтапного отката к предыдущим состояниям (в пределах истории изменений).

__Примечание:__ Если с момента последнего сохранения не было выполнено действий, отмена будет недоступна.

### Повторить
Данная функция позволяет повторить последнее действие с текстом.

При нажатии кнопки __"Повторить"__ программа повторяет последнее действие, выполненное пользователем. Это может быть повтор вставки, удаления символа или другого изменения текста.

Функцию можно использовать несколько раз подряд.

### Вырезать
Данная функция позволяет удалить выделенный текст из документа и поместить его в буфер обмена.

При нажатии кнопки __"Вырезать"__ выбранный фрагмент текста будет удалён из текущего документа и сохранён в буфере обмена. Этот текст можно будет вставить в любое место текущего или другого документа с помощью функции __"Вставить"__.

__Примечание:__ Если текст не выделен, команда __"Вырезать"__ будет недоступна.

### Копировать
Данная функция позволяет скопировать выделенный текст в буфер обмена без удаления его из документа.

При нажатии кнопки __"Копировать"__ выбранный фрагмент текста сохраняется в буфере обмена. После этого его можно вставить в любое место текущего или другого документа с помощью функции __"Вставить"__.

__Примечание:__ Если текст не выделен, команда __"Копировать"__ будет недоступна.

### Вставить
Данная функция позволяет вставить текст из буфера обмена в текущий документ.

При нажатии кнопки __"Вставить"__ содержимое буфера обмена будет добавлено в то место документа, где находится курсор. Если текст был ранее скопирован или вырезан, он будет вставлен в выбранную область.

__Примечание:__ Если буфер обмена пуст или не содержит текстовых данных, команда __"Вставить"__ будет недоступна.

### Удалить
Данная функция позволяет удалить выделенный текст из документа без сохранения его в буфере обмена.

При нажатии кнопки __"Удалить"__ выбранный фрагмент текста будет удалён из текущего документа. В отличие от функции __"Вырезать"__, удалённый текст не сохраняется в буфере обмена и не может быть вставлен повторно с помощью команды __"Вставить"__.

__Примечание:__ Если текст не выделен, команда __"Удалить"__ будет недоступна.

### Выделить все
Данная функция позволяет выделить весь текст в текущем документе.

При нажатии кнопки __"Выделить все"__ весь текст в документе будет автоматически выделен. Это позволяет легко выполнить действия с текстом, такие как копирование, вырезание или удаление, применив их ко всему содержимому.

## Справка

### Вызов справки
Данная функция позволяет получить доступ к справочной информации о программе.

При нажатии кнопки __"Вызов справки"__ откроется окно справки, где пользователю будут представлены подробные инструкции и описание всех функций программы. Это поможет лучше понять, как использовать программу и её возможности.

Справка включает описание всех доступных команд, а также часто задаваемые вопросы и советы по решению возможных проблем.

### О программе
Данная функция позволяет получить информацию о текущей версии программы.

При нажатии кнопки __"О программе"__ откроется окно с информацией о разработчике программы, версии приложения, а также с контактными данными и ссылками на ресурсы поддержки.

В окне также будет указана информация о лицензии на использование программы и ее авторских правах.
